The Smartcard reader example is implemented as a USB CCID device. The source code could be found in the following
path : 
       "..\Samples\CardReader\CCID".

User can build and download the code to SmartCard reader demo board. Once the board is attached to Windows PC, it
will be recognized as a card reader.

Two application notes are provided to help to read the example source code. These two application notes could be 
found in the following path :
       "..\Samples\CardReader\Doc".

"AN1025EN An Example of USB CCID Class V1.00.pdf" - Introduction to the implementation of USB CCID
"AN1030EN Smart Card Reader Example V1.00.pdf" - Introduction to the handling process of APDU protocol and 
                                                 smartcard host controller.
